# 증착 공정
웨이퍼 위에 분자 또는 원자 단위의 매우 얇은 막(박막, Thin Film)을 입히는 과정이다. (박막 공정이라고도 부른다)

* 목적: 회로 간 연결(도체), 신호 간섭 차단(절연), 외부 충격 방지(보호) 등 다양한 기능을 수행한다.
* 중요성: 반도체가 미세화되면서 산화 공정보다 더 얇고 정밀한 막을 형성할 수 있는 증착 기술이 필수적이게 되었다.

# 증착의 주요 수치(품질 지표)
* 균일도(Uniformity): 웨이퍼 전체영역에 걸쳐 막의 두께가 얼마나 일정한지 나타낸다.
* 스텝 커버리지(Step Coverage): 요철이 있는 표면에서 상단, 측면, 하단부의 막질 두께가 얼마나 균일하게 형성되는지 보여준다.
* 갭 필(Gap Fill): 회로 사이의 좁고 깊은 빈 공간(Gap)을 빈틈없이 채우는 능력. 채우지 못하면 내부에 빈 공간이 생겨 구조적 결함이 된다.

# 증착의 종류 비교
 | 구분 | 물리적 기상 증착 (PVD) | 화학적 기상 증착(CVD) | 원자층 증착(ALD) |
 | ------- | ------- | -------| ------- |
 | 원리 | 물질을 기화시켜 직접 달라붙게 함(스퍼터링 등) | 가스의 화학 반응으로 입자를 형성해 증착  | 원자 층을 한 층씩 번갈아 가며 쌓음 |
 | 장점 | 불순물이 적고 순도가 매우 높음 | 공정 속도가 빠르고 스텝 커버리지가 우수함 | 최고 품질, 원자 단위의 두께 제어 가능 |
 | 용도 | 금속 배선(알루미늄, 구리 등) | 절연막, 하드 마스크 등 범용 | D램 커패시터 등 초미세 공정 |

# 이온주입 공정
순수한 실리콘 웨이퍼(부도체)에 불순물(이온)을 넣어 전기적 성질을 갖게 만드는 과정

* N형 반도체: 15족 원소 (인, 비소 등) 주입
* P형 반도체: 13족 원소 (붕소 등) 주입
* 특징: 이온을 미세 가스 입자로 만들어 원하는 깊이만큼 균일하게 주입한다.
