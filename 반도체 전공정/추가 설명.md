# 8대 공정 흐름
웨이퍼 준비 -> 산화 -> 포토 -> 식각 ->  증착 -> 금속배선 -> 반복 -> 전기적 특성 확보

각 과정은 독립된게 아닌, 앞 공정 편차가 뒤 공정 수율에 누적 영향을 준다.

# 각 공정별 핵심 변수
1. 산화 공정
   * 온도
   * 시간
   * O₂ / H₂O 유량
   * 압력
   * 웨이퍼 위치
     - 두께 편차 = Vth 변동 -> 수율 영향
    
2. 포토 공정
   * 노광 에너지(Dose)
   * Focus
   * PR 두께
   * Alignment / Overlay 오차
   * CD(Critical Dimension)
     - CD variation = 전류 특성 변화

3. 식각 공정
   * Rf power
   * Gas ratio
   * Pressure
   * Time
   * Chuck temperature
     - 과식각/미식각 -> 패턴 붕괴 -> 누설 전류
    
4. 증착 공정
   * 온도
   * 전구체 유량
   * Pressure
   * 증착 시간
   * Plasma power
     -> Step coverage 불량 -> 단선/void

5. 금속배선 공정
   * Dose
   * Energy
   * Beam current
   * Tilt angle
     -> 농도 편차 -> Threshold voltage shift

# 수율과 연결
수율은 단일 공정 문제가 아닌 공정 편차 누적의 결과이다.

예를들어, 산화 두께 + 식각 CD 편차 -> 전류 특성 변화 혹은 증착 void -> 금속 단선 -> Open 불량이 된다.

공정 변수 -> 물리 변화 -> 전기 특성 변화 -> 수율(결과)
