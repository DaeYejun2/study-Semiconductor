# 8대 공정 흐름
웨이퍼 준비 -> 산화 -> 포토 -> 식각 ->  증착 -> 금속배선 -> 반복 -> 전기적 특성 확보

각 과정은 독립된게 아닌, 앞 공정 편차가 뒤 공정 수율에 누적 영향을 준다.

# 각 공정별 핵심 변수
## 1. 산화 공정
   ### 핵심 변수
   * 온도
   * 시간
   * O₂ / H₂O 유량
   * 압력
   * 웨이퍼 위치

   ### 전기적 불량
   * 산화막 두께 편차 -> Vth shift
   * Gate oxide 과박 -> Leakage 증가
   * 과산화 -? Drive current 감소
  
   ### 수율 연관성
   Gate 특성 불균일  -> Cell 동작 불안정 -> Parametric fail 증가 -> 수율 하락
    
## 2. 포토 공정
   ### 핵심 변수
   * 노광 에너지(Dose)
   * Focus
   * PR 두께
   * Alignment / Overlay 오차
   * CD(Critical Dimension)
     
   ### 전기적 불량
   * CD 감소 -> 채널 폭 감소 -> 저항 증가
   * CD 증가 -> Short 위험
   * Overlay 불량 -> Gate misalignment -> 누설
  
   ### 수율 연결
   패턴 오차 -? 전기 특성 편차 -> Bit fail / Open / Short 증가

## 3. 식각 공정
   ### 핵심 변수
   * Rf power
   * Gas ratio
   * Pressure
   * Time
   * Chuck temperature
   
   ### 전기적 불량
   * 과식각 -> Gate oxide 손상 -> Leakage
   * 미식각 -> 잔여막 -> Short
   * Sidewall 손상 -> 신뢰성 저하
  
   ### 수율 연결
   Gate stack 손상 -> Retention fail / Leakage 증가 -> 수율 하락
    
## 4. 증착 공정 (CVD / ALD / PVD)
   ### 핵심 변수
   * 온도
   * 전구체 유량
   * Pressure
   * 증착 시간
   * Plasma power
   
   ### 전기적 불량
   * Step coverage 부족 -> Void 발생
   * 두께 편차 -> 저항 증가
   * 박리 -> Open
  
   ### 수율 연결
   배선 단선 / 접촉 저항 증가 -> Functional fail

## 5. 이온 주입
   ### 핵심 변수
   * Dose
   * Energy
   * Beam current
   * Tilt angle

   ### 전기적 불량
   * 과도한 dose -> Leakage 증가
   * 부족한 dose -> Vth 상승
   * Energy 편차 -> Junction depth 이상
  
   ### 수율 연결
   cell threshold 분포 확대 -> Program/Erage 실패 증가

## 6. 금속 배선 공정
   ### 핵심 변수
   * 두께
   * CMP 균일도
   * Barrier 형성 품질
   * Anneal 조건
  
   ### 전기적 불량
   * EM 취약 -> 신뢰성 저하
   * Void -> Open
   * 금속 확산 -> Leakage
  
   ### 수율 연결
   Open/Short 증가 -> 최정 테스트 Fail 확률 증가


   

# 수율과 연결
수율은 단일 공정 문제가 아닌 공정 편차 누적의 결과이다.

예를들어, 산화 두께 + 식각 CD 편차 -> 전류 특성 변화 혹은 증착 void -> 금속 단선 -> Open 불량이 된다.

공정 변수 -> 물리 변화 -> 전기 특성 변화 -> 수율(결과)
